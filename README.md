# מעקב משקל ⚖️

אפליקציית ווב פשוטה (קובץ HTML יחיד) למעקב משקל יומי וחישוב ממוצע שבועי.

## איך משתמשים

פותחים את `index.html` בדפדפן — זהו. אין צורך בהתקנה, בשרת או בחיבור לאינטרנט.

כדי שיהיה נוח, אפשר להתקין כקיצור דרך על הטלפון/שולחן העבודה דרך תפריט הדפדפן ("Add to Home Screen" / "התקן אפליקציה").

## מה היא עושה

- **הזנת משקל יומי** — שדה משקל + בחירת תאריך. שבת חסומה (לא ניתן להזין).
- **ממוצע שבועי** — שבוע מוגדר כראשון–שישי (6 ימים). הממוצע מחושב לפי מספר הימים שהוזנו בפועל (אם הוזנו 4 ימים, מחלקים ב-4). ביום שישי מוצג הממוצע כסופי; בשאר הימים מוצג כממוצע זמני עם מונה ימים.
- **הפרדה אוטומטית בין שבועות** — כל הזנה משויכת אוטומטית לשבוע הנכון לפי התאריך.
- **היסטוריה שבועית** — רשימת שבועות עבר עם הממוצע, הפרש מהשבוע הקודם (▼/▲), וגרף קו של מגמת המשקל לאורך זמן.
- **עריכת עבר** — לחיצה על שבוע בהיסטוריה פותחת את ששת ימיו, וניתן לערוך או למחוק כל יום.
- **גיבוי ושחזור** — ייצוא כל הנתונים לקובץ JSON, וייבוא בחזרה (כולל העברה בין מכשירים).

## איפה נשמרים הנתונים

הנתונים נשמרים ב-`localStorage` של הדפדפן במכשיר שבו משתמשים — מקומית, פרטית, ללא שרת. הם נשארים גם לאחר סגירת הדפדפן.

> **חשוב:** ניקוי "נתוני גלישה" אגרסיבי בדפדפן עלול למחוק את הנתונים. מומלץ לייצא קובץ גיבוי מדי פעם.

## גיבויים

תיקיית `backups/` מיועדת לקבצי הגיבוי שמיוצאים מהאפליקציה. אפשר לשמור שם עותקים ולדחוף לגיט כדי לשמור היסטוריה מגובה.
